## GitHub Secrets
Settings > Secrets and variables > Actions > New repository secret 에 아래 2개를 추가하세요.

- TELEGRAM_BOT_TOKEN
- TELEGRAM_CHAT_ID

## 자동 실행 시간
.github/workflows/fund.yml 기준 한국시간 매일 17:10 실행입니다.

## 수동 실행
Actions > Samo Fund Auto Capture > Run workflow
